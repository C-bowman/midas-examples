# midas-examples
Code examples for [MIDAS](https://github.com/C-bowman/midas), a Python framework for multi-instrument Bayesian data analysis.

### Z-effective profile inference example
[Click here for a jupyter notebook example](https://github.com/C-bowman/midas-examples/blob/main/z-eff/z_effective_inference.ipynb)
which demonstrates how the profile of Z-effective can be inferred by combining
Thomson-scattering and Bremsstrahlung measurements.
